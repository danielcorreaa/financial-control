Compile e verifique o frontend deste projeto.

Passos:
1. Entre na pasta `frontend/`
2. Execute `npm run build` (que roda `tsc && vite build`)
3. Reporte quaisquer erros de TypeScript ou de build encontrados
4. Se o build for bem-sucedido, informe o tamanho do bundle gerado em `frontend/dist/`
5. Se houver erros, analise-os e sugira correções
